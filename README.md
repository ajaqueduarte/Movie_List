# Razor Pages Movies

## Overview
Razor Pages Movies is a basic web application developed with ASP.NET Razor Pages, demonstrating CRUD (Create, Read, Update, Delete) operations for managing a collection of movies.

## Features
- Create, Read, Update, and Delete movie entries
- Responsive UI design using Tailwind CSS with DaisyUI components
- Utilizes ASP.NET Core for backend logic and Razor Pages for frontend rendering
- Integrated with a local database for storing movie information
